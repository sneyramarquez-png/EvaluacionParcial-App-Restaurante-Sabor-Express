# EvaluacionParcial-App-Restaurante-Sabor-Expressa
## Ceviche Express es una aplicación móvil desarrollada en Android que permite a los usuarios visualizar un plato de ceviche, conocer su precio y realizar un pedido ingresando su dirección de entrega. La app presenta una interfaz simple, moderna y amigable, facilitando la experiencia de compra rápida desde el celular.

<img width="313" height="672" alt="image" src="https://github.com/user-attachments/assets/1cd46d49-016a-4203-ba83-a91365e90130" />


La interfaz principal muestra una imagen del plato de ceviche, el nombre del producto en texto grande y color rojo, el precio (S/ 30.00), 
un campo para ingresar la dirección de entrega y dos botones: “Ordenar” y “Ver Menú”. El diseño utiliza colores llamativos como rojo, verde y
morado para resaltar la información y diferenciar las acciones principales de la aplicación.

<img width="333" height="566" alt="image" src="https://github.com/user-attachments/assets/eaea8da3-8f2e-4f22-a3cd-08e6b2785dca" />

Se trabajó sobre un layout en blanco (probablemente un ConstraintLayout o LinearLayout), donde se agregaron los elementos manualmente,esta plantilla permitió organizar los componentes de manera vertical y centrada.
